# Requirements - KayyDiang E-Learning Platform

## v1 Requirements

### Authentication
- [ ] **AUTH-01**: Utilisateur peut creer un compte avec email/password
- [ ] **AUTH-02**: Utilisateur peut se connecter avec email/password
- [ ] **AUTH-03**: Utilisateur peut se deconnecter
- [ ] **AUTH-04**: Utilisateur peut reinitialiser son mot de passe via email
- [ ] **AUTH-05**: Utilisateur reste connecte entre les sessions

### Courses
- [ ] **COURSE-01**: Catalogue des cours avec recherche et filtres
- [ ] **COURSE-02**: Details d'un cours (description, curriculum, instructeur)
- [ ] **COURSE-03**: Chapitres et lecons d'un cours
- [ ] **COURSE-04**: Lecture video avec controles
- [ ] **COURSE-05**: Suivi de progression par lecon
- [ ] **COURSE-06**: Inscription a un cours

### Quizzes
- [ ] **QUIZ-01**: Questions a choix multiples
- [ ] **QUIZ-02**: Score apres completion
- [ ] **QUIZ-03**: Resultat reussi/echoue selon seuil

### Assignments
- [ ] **ASSIGN-01**: Soumission de fichier pour un devoir
- [ ] **ASSIGN-02**: Historique des soumissions

### Dashboard
- [ ] **DASH-01**: Pourcentage de progression global
- [ ] **DASH-02**: Temps passe a apprendre

### Certificates
- [ ] **CERT-01**: Certificat a la completion d'un cours

### Notifications
- [ ] **NOTIF-01**: Notifications email (inscription, deadlines, notes)

### Instructor Tools
- [ ] **INST-01**: Creation de cours via assistant
- [ ] **INST-02**: Ajout de chapitres et lecons
- [ ] **INST-03**: Publication d'un cours

## Out of Scope

- **OAuth tiers**
- **Forum/Discussion**
- **Application mobile native**
- **Fonctionnalites IA/ML**
- **Paiement integre**
- **Streaming live**
